Постановка задачи

Конструкция enum используется в языке Swift для определения перечислений.

В рамках данного варианта рассматривается объявление перечисления с набором кейсов.

Формат записи: "enum имя_перечисления { case имя_кейса1; case имя_кейса2; ... };"

Где:

  1. имя_перечисления начинается с буквы и может содержать буквы, цифры и символ '_';

  2. каждый кейс начинается с ключевого слова case, за которым следует идентификатор (имя кейса);

  3. после каждого кейса обязательно ставится точка с запятой ;;

  4. тело перечисления заключается в фигурные скобки { };

  5. вся конструкция завершается точкой с запятой ; после закрывающей скобки.

Примеры корректных записей:

  1. enum Day { case monday; case tuesday; case wednesday; };

  2. enum Status { case active; case inactive; case pending; };

  3. enum Direction { case north; case south; case east; case west; };
